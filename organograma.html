Sistema de Organograma

// O Desafio

Transformar informações dispersas em múltiplas planilhas de Excel em uma visualização clara, dinâmica e acessível. A ferramenta interpreta listas extensas de cargos e supervisões, convertendo-as em uma estrutura organizada e pronta para download em segundos.

// Engenharia do Projeto

Aprofundei a manipulação de dados para automatizar a interpretação da hierarquia corporativa. O fluxo elimina o ajuste manual: o usuário insere os dados e recebe imediatamente um organograma formatado, ajustável e funcional.

O resultado é uma interface moderna que facilita análises estruturais e apresentações executivas, elevando o padrão de como dados hierárquicos são processados.
